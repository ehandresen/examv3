Races API Documentation

Get All Races

Retrieve a list of all races.

GET /api/races

Get Race by ID

Retrieve information about a specific race by providing its ID.

GET /api/races/{id}

Get Race by Grand Prix

Retrieve information about a race by providing part of its Grand Prix name.

GET /api/races/GetByGrandPrix/{grandPrix}

Get Races with Max Laps

Retrieve races with a number of laps equal to or less than the specified value.

GET /api/races/GetMaxLaps/{maxLaps}

Add a New Race

Add a new race by providing the necessary details in the request body.

POST /api/races

Update a Race

Update an existing race by providing the updated details in the request body.

PUT /api/races

Delete a Race

Delete a race by providing its ID.

DELETE /api/races/{id}
